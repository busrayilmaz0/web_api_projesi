# web_api_projesi
Bu Web API projesi, şirket çalışanları için departman bazlı bir yetkilendirme sistemidir. Kullanıcılar TC kimlik, ad, soyad, kullanıcı adı ,şifre, departman gibi bilgilerle veritabanına kaydolur. Giriş yapıldığında, Thread ve Regedit yapıları sayesinde kullanıcının departmanına özel (örn: bilgisayar mühendisi için cmd) .exe dosyaları listelenir.
